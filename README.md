## About This Archive

This is a snapshot-based archive containing the latest state of each project at the time of archival:

- **Build artifacts removed**: `node_modules`, `__pycache__`, `.cache`, `dist`, `build`
- **Secrets redacted**: API keys, tokens, and credentials have been removed or replaced
- **Binary assets removed**: Large game files, GeoIP databases, and media assets
- **Lock files removed**: `package-lock.json`, `pnpm-lock.yaml`, etc.

All projects are old, deprecated, and inactive. Individual project READMEs are preserved where they existed.
