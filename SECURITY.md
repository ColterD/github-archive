# Security Policy

## Archived Repository

**This repository is archived and no longer maintained.**

All projects in this archive are deprecated, unmaintained, and preserved solely for historical reference. They are not intended for production use.

## Reporting Vulnerabilities

**No security reports will be accepted or addressed.**

Since this is an archived repository:

- No security patches will be released
- No dependency updates will be made
- No pull requests will be merged
- No issues will be triaged

## Known Vulnerabilities

This archive contains old code with known security vulnerabilities in dependencies. This is expected and will not be fixed. The code is preserved as-is for reference purposes only.

**Do not use any code from this repository in production environments.**

## If You Find Exposed Secrets

If you discover any exposed secrets (API keys, tokens, passwords) that were missed during the archival cleanup process, please contact the repository owner directly rather than opening an issue. However, note that any secrets found in this archive should be considered compromised and already rotated.
