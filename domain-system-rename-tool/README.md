# Domain System Rename Tool

> **Archived** - This project is deprecated and unmaintained.

PowerShell GUI tool for renaming Windows computers on a domain.

## Files

- `name-change-gui.ps1` - Main PowerShell script with GUI interface

## Usage

Run the script with administrator privileges:

```powershell
.\name-change-gui.ps1
```
